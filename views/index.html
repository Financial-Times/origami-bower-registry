Install Financial Times GitHub repositories as Bower components

The Origami Bower Registry is a Bower-compatible JSON API which lists all public repositories owned by the Financial Times.

To use the Origami Bower registry, update your ~/.bowerrc file to the following: 

{
	"registry": {
		"search": [
			"https://origami-bower-registry.ft.com",
			"https://bower.herokuapp.com"
		]
	}
}
